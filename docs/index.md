# ProtectKids

## 📌 Sobre o projeto
O ProtectKids é uma plataforma desenvolvida na disciplina de Métodos de Desenvolvimento de Software (MDS) da Universidade de Brasília (UnB).

A proposta do sistema é monitorar dados legislativos relacionados à proteção de crianças e adolescentes no ambiente digital, organizando essas informações de forma estruturada para facilitar análises e apoiar a tomada de decisão em políticas públicas.

## 🎯 Objetivo
- Centralizar informações legislativas relevantes
- Facilitar o acompanhamento de propostas e mudanças na legislação
- Apoiar análises sobre proteção infanto-juvenil no ambiente digital
- Auxiliar na tomada de decisão por meio de dados estruturados
